## 🧰 System Requirements

* Xcode Editor

* iPhone 11 or latest version

## 🛠️ Installation

#### Open in Xcode, go to source control -> clone -> enter repository URL.

```
https://github.com/Padman83/iOS-Hallowen-Emoji-s.git
```

#### Or Use the Terminal.

```
git clone https://github.com/Padman83/iOS-Hallowen-Emoji-s.git
```
